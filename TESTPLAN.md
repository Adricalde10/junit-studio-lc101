#  BalancedBrackets

### 1. Casos Básicos

#### Caso 1: Cadena vacía
- Entrada: ""
- Salida Esperada: Verdadero (true)
- Comentarios: Una cadena vacía debe considerarse balanceada.

#### Caso 2: Solo corchetes simples
- Entrada: "[]", "{}", "()"
- Salida : Verdadero (true)
- Comentarios: Solo corchetes simples deben devolver verdadero.
